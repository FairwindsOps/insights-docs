# RBAC

The RBAC tab shows you a list of Roles and ClusterRoles in your cluster. It ranks these by their level of access:

These roles are able to access secrets or edit RBAC permissions.

These roles are able to make changes to the cluster.

These roles are only able to read resources that aren't secrets.

Roles that don't have any associated bindings are treated as being safe, and will appear below any roles with bindings.
